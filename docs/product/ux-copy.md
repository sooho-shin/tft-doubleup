- 티어
- 목적

- 디스코드

- 모집 상태

MVP에서는 `성별`을 필터로 제공하지 않는다. 성별은 방 만들기와 신청에서 본인 정보로만 입력한다.

## 서버 문구

- 한국 서버
- KR 고정

MVP에서는 서버 선택 필터를 제공하지 않는다. 화면에 서버를 표시해야 할 때는 `KR` 또는 `한국 서버`로 고정 표시한다.

## 디스코드

- 대화 가능
- 듣기만 가능
- 디스코드 불가능

필드 타이틀은 `디스코드`로 쓴다. 선택지는 위 3개만 사용한다.

쓰지 않는 문구:

- Discord 가능
- 음성 필수
- 음성 선택
- 음성 없이 가능

쓰지 않는 문구:

- 오늘 가능
- 이번 주 가능
- 시간 조율
- 예약

## 성별

- 본인 성별
- 남성
- 여성
- 비공개

규칙:

- 기본값은 `비공개`로 둔다.
- `상대 성별`, `원하는 성별`, `성별 필터` 문구는 MVP에서 쓰지 않는다.
- 성별은 매칭 점수나 자동 추천 기준으로 표현하지 않는다.

## 준비중

- 준비중입니다.
- MVP에서는 아직 제공하지 않는 기능입니다.
- 지금은 모집방에서 바로 가능한 파트너를 찾아보세요.

## 빈 상태

- 조건에 맞는 모집방이 없습니다.
- 필터를 줄이거나 `지금 가능`을 꺼보세요.
- 아직 받은 신청이 없습니다.
- 프로필을 완성하면 더 잘 맞는 파트너를 찾기 쉽습니다.

## 폼 라벨

- 방 제목
- 내 더블업 티어
- 원하는 티어 범위
- 본인 성별
- 원하는 파트너
- 선호하는 운영 방식
- 디스코드
- 한 줄 설명
- 상세 설명
- 신청 메시지

## 방 만들기

- 어떤 파트너를 찾고 있나요?
- KR 서버 고정
- 지금 바로 같이 할 수 있는 방만 만들 수 있습니다.
- 예약 시간과 시간 조율은 MVP에서 제공하지 않습니다.
- 방이 생성되었습니다.
- 방을 만들 수 없습니다. 입력값을 확인해 주세요.

쓰지 않는 문구:

- 모집글 작성
- 게시글 작성
- 글쓰기
- 예약 방 만들기
- 시간 조율 방 만들기

## 문구 톤

- 라벨은 짧고 빠르게 읽히게 쓴다.
- 한국 TFT 유저가 알아보는 게임 용어를 쓴다.
- 사용자가 캐주얼한 커뮤니티 톤을 명시적으로 원하지 않는 한 과한 유행어나 장난스러운 표현은 피한다.
- 설명형 문장보다 행동 중심 라벨을 우선한다.
