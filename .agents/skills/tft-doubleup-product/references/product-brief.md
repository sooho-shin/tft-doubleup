# TFT 더블업 팀원 찾기 제품 브리프

## 포지셔닝

TFT 더블업 팀원 매칭 서비스는 TFT 전적/메타 도구처럼 신뢰감 있고 정보 밀도 높은 서비스로 만든다. 일반 커뮤니티나 자유게시판보다 랭크 로비와 메타 대시보드에 가까워야 한다.

핵심 약속:

> 나와 맞는 더블업 파트너를 빠르게 찾고, 안 맞는 팀원을 만날 확률을 줄인다.

## 주요 사용자

- 더블업 랭크를 올리고 싶은 유저
- 지금 바로 같이 큐를 돌릴 파트너를 찾는 유저
- 즐겜/빡겜 성향이 맞는 팀원을 찾는 유저
- 초보나 복귀 유저와 편하게 할 파트너를 찾는 유저

## 핵심 사용자 행동

1. 지금 가능한 사람을 확인한다.
2. 한국 서버(KR) 안에서 티어, 디스코드 대화 방식, 목적, 플레이 스타일로 필터링한다.
3. 모집 카드만 보고 대략적인 적합성을 판단한다.
4. 신청하거나 수락 후 연락 정보를 확인한다.
5. 다음 매칭에도 쓸 수 있는 프로필을 유지한다.

## MVP 범위

반드시 필요:

- 모집글 목록
- 한국 서버(KR) 고정
- 지금 가능 중심 목록
- 티어 범위, 디스코드, 목적, 플레이 스타일 필터
- 모집글 상세
- 사용자 프로필
- 신청/수락 흐름
- 모집 상태: 모집중, 매칭완료, 마감

다음 단계:

- JP, NA, EUW 등 다중 서버 지원
- 예약 모집, 시간 조율, 미래 시간대 기반 매칭
- Riot ID 인증
- Riot API 기반 더블업 티어 동기화
- 파트너 추천 점수
- 파트너 추천 화면
- 랭킹 화면
- 가이드 화면
- 응답률과 최근 활동
- 게임 후 매너 후기
- 서비스 내 채팅

MVP에서 제외:

- 범용 소셜 피드
- 복잡한 길드/팀 시스템
- 큰 마케팅 랜딩페이지
- 검증되지 않은 랭킹/실력 보장 표현
- 예약제 모집
- 오늘 가능/이번 주 가능/시간 조율 같은 미래 시간대 모집
- 실제 파트너 추천 기능
- 실제 랭킹 기능
- 실제 가이드 콘텐츠

MVP에서는 `파트너 추천`, `랭킹`, `가이드` 메뉴를 네비게이션 진입점으로만 둘 수 있다. 사용자가 누르면 실제 화면으로 이동하지 않고 `준비중입니다.` 상태 또는 준비중 모달을 보여준다.

## 매칭 적합성 신호

중요도 높음:

- 비슷한 티어 또는 티어 차이를 허용한다는 명시
- 같은 플레이 목적
- 디스코드 대화 방식 일치
- 지금 같이 플레이할 수 있는 상태

중요도 중간:

- 플레이 스타일 호환
- 소통 방식 일치
- 최근 활동
- 응답률

중요도 낮음:

- 긴 자기소개
- 프로필 장식
- 단순 인기

MVP에서는 모든 모집글과 사용자가 한국 서버(KR)를 전제로 하므로 `같은 서버`는 매칭 점수 신호가 아니라 기본 조건이다.
MVP에서는 예약제가 없으므로 `플레이 시간대 겹침`은 매칭 점수 신호가 아니다. 지금 가능한지 여부가 기본 조건이다.

## 모집 카드 우선순위

카드는 상세 페이지를 열지 않아도 다음 정보를 보여줘야 한다.

1. KR 서버와 티어
2. 지금 가능 여부
3. 목적
4. 디스코드 대화 방식
5. 플레이 스타일 태그
6. 짧은 제목
7. 신청 버튼

예시:

```text
KR · Emerald II · 지금 가능
랭크 같이 올릴 파트너 구합니다
대화 가능 · 순방 위주 · 바로 2판 가능
#운영덱 #소통선호 #리롤가능
신청하기
```
